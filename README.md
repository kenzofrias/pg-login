<h1>Projeto Página de Login</h1>
<p>Este é um projeto totalmente autoral - baseado numa página de login visitada anteriormente - visando aperfeiçoar meus conhecimentos em HTML5 e CSS3, mas especificamente na parte de Media Queries.</p>

<h2>Sobre o projeto</h2>
<p>
  A página em questão está bem contruída, com os campos de preenchimento em login bem desenhados e organizados. 
  Além de que há a presença de links para cadastro ou "esqueci senha" (no momento em direcionamentom, já que são para testes iniciais). Gostei do desafio, já que apesar de simples, por trás é um pouco trabalhoso.
</p>

<h2>Visão sobre o projeto</h2>
<p>
  Tive algumas dificuldades, mas não me abalei, procurei soluções que podem não ser as ideais para o momento, mas que funcionaram. Espero poder estudar e compreender falhas neste exercício cometidas. 
  Agora é foco no próximo projeto.
